# PhD Hub

This is a GitHub-hosted recreation of your Notion-style PhD template interface.

## Open the interface
- [`index.html`](./index.html)

## Edit without CLI
1. Open any page from the dashboard.
2. Click **Edit this file** in GitHub.
3. Update content.
4. Commit directly in GitHub.
